All notable changes to **Nightshade Theme** are documented here.

## [Unreleased]

## [1.0.2] – 2026-03-28

### Changed
- Broadened IDE compatibility metadata to support the full 2025.2 build line.

## [1.0.1] – 2026-03-25

### Fixed
- Renamed `Nightshade.xml` to `nightshade.xml` to ensure case-sensitive filesystem compatibility.

### Changed
- Updated release workflow configuration (`release.yml`).

## [1.0.0] – 2026-03-25

### Added
- Initial release of Nightshade Theme with deep purple and elegant dark interface.
- Optimized syntax highlighting for improved code readability.
- Designed for all JetBrains IntelliJ IDEs (2025.2+).
- Marketplace-compliant description with clickable screenshot preview.
- Optimized theme assets and thumbnail preview for better preview experience.
- Polished HTML markup for strict Marketplace compatibility.
- Comprehensive README with installation and feature documentation.
